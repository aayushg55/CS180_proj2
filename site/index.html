Part 1: Fun with Filters

Part 1.1: Finite Difference Operator

The image derivatives are computed by convolving the finite difference operators D_x and D_y with the image. That is, img_grad_x = img * D_x and img_grad_y = img * D_y, where * is the convolution operator. The magnitude of the gradient is computed by taking the per pixel magnitude of the derivatives in x and y, so img_grad_magnitude = sqrt(img_grad_x^2 + img_grad_y^2). To turn this into an edge image, I binarize the gradient magnitudes with thresholding.

Part 1.2: Derivative of Gaussian (DoG) Filter

The gradient magnitudes produced by directly applying the finite difference operators are quite noisy. As such, we first apply a blurring operator — a gaussian — and then convolve the derivatives with this blurred image.

T​​he resulting gradient magnitude image produced by starting with a blurred image has significantly less noise, making it easier to threshold. This is because by applying a Gaussian filter first, we reduce the high frequencies (low pass filter). The threshold can now be lowered to 0.04 from 0.21. Additionally, the edges produced are more prominent and thicker.

We can get the same effect with a single convolution instead of two by creating ​derivatives of gaussian filters from convolving the gaussian with D_x and D_y. We can see this results in the same final gradient magnitudes as before.

Part 2: Fun with Frequencies!

Part 2.1: Image "Sharpening"

Image sharpening is implemented using the unsharp mask. This is done by subtracting a blurred version of the image from the original image to get the high frequencies of the image. We then add these high frequencies scaled by some alpha to the original image to "sharpen" it.

Part 2.2: Hybrid Images

The basic idea is that high frequency tends to dominate perception when it is available, but, at a distance, only the low frequency (smooth) part of the signal can be seen. By blending the high frequency portion of one image with the low-frequency portion of another, we get a hybrid image that leads to different interpretations at different distances. To get the low frequencies, we can convolve the image with a gaussian, and to get the high frequencies, we can subtract the original image from these low frequencies.

Part 2.3: Gaussian and Laplacian Stacks

To create a gaussian stack, I take the original image and repeatedly convolve it with a gaussian filter. Each successive image in the stack consists of lower and lower frequencies. To create a Laplacian stack, I take the difference of consecutive image in the gaussian stack. The final image in the stack consists of the lowest frequencies. The Laplacian stack thus consists of bandpass images. Collapsing the Laplacian stack, or adding together all the images, reconstructs the original image. I extend this to color for all examples by processing each channel separately.

Part 2.4: Multiresolution Blending

To blend images, we first need a gaussian stack of the mask which represents how the images will be merged. We can then multiply our Laplacian stacks of the images with their corresponding versions of the mask to create intermediate masked stacks. We can then collapse both intermediate stacks and sum them to get the final result.
